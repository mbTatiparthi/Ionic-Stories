Ion Stories - RESTful API

Ionic stories app for Android

List Categories

GET https://ionstories.local/restapi.php?api=categories

Parameters

ParameterValueDescriptionData Type
sortasc | ascOrder sort attribute ascending or descendingString
orderbyid |
category-name |
category-image |
random		Sort collection by object attributeString
idLimit result set to items with more specific by `id`Integer
category-nameLimit result set to items with more specific by `category_name`String
category-imageLimit result set to items with more specific by `category_image`String

Example

- https://ionstories.local/restapi.php?api=categories
- https://ionstories.local/restapi.php?api=categories&orderby=id
- https://ionstories.local/restapi.php?api=categories&orderby=random
- https://ionstories.local/restapi.php?api=categories&orderby=id&sort=desc



Retrieve a Category 

GET https://ionstories.local/restapi.php?api=categories&id=<id>

Parameters

ParameterValueDescriptionData Type
idUnique identifier for the object and use random for random IDsInteger

Example

- https://ionstories.local/restapi.php?api=categories&id=1
- https://ionstories.local/restapi.php?api=categories&id=2
- https://ionstories.local/restapi.php?api=categories&id=random
